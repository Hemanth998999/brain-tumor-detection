# 🧠 Brain Tumor Detection using EfficientNetB4

An AI-powered web application that detects brain tumor types from MRI scan images using **Deep Learning (EfficientNetB4)** and **Streamlit**.

🔗 **Live Demo:** https://brain-tumor-detection-hemanth.streamlit.app/

---

## 📌 Project Overview
This project uses a **Convolutional Neural Network based on EfficientNetB4** to classify MRI brain scan images into four categories:

- Glioma
- Meningioma
- Pituitary
- No Tumor

The application allows users to upload MRI images and instantly get:

- ✅ Predicted tumor type
- 📊 Confidence score
- 📈 Class probability distribution
- 🌐 Real-time web deployment

---

## 🚀 Features
- 🧠 Brain tumor classification from MRI images
- 📤 Image upload interface
- 📊 Confidence percentage display
- 📈 Probability bars for all classes
- ⚡ Fast inference using EfficientNetB4
- 🌐 Deployed using Streamlit Cloud
- 🔁 Supports multiple image predictions
